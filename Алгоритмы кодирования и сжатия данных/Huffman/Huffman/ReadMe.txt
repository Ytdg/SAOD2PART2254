Кодирование-декодирование по Хаффману.
Проект в задание (убрать реализацию DecompressFile).

Управляется опеределением #define WITH_BLOCKS
Если определено WITH_BLOCKS то собирает вариант с двумя типам блоков Huffman и RLE.
Результат представляется в виде последовательности блоков 2-х типов.

В каталоге Data имеются файлы для отладки, тестирования и оценки эффективности сжатия.
0....2.txt для тестирования
51.txt и h.exe - для оценки  сжатия. Текстовый и бинарный.

(51.txt фактически RFC 1951.txt)
